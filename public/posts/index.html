Como criar Máquinas Virtuais do Azure  [draft]

No Azure, o primeiro passo no processo para criar uma máquina virtual irá requerer uma assinatura disponível. Além disso, junto aos demais passos posteriores, você determinará as características para os componentes da máquina virtual que estão sempre presentes. Tais componentes são descritos abaixo: Resource Group Uma Máquina Virtual é disponibilizada para um Resource Group, no qual contém uma região do Azure atrelada a ele. Isso significa que você terá uma máquina física naquele local com propósito de aproximar consumo dos usuários para as aplicações e serviços hospedados naquela máquina....
